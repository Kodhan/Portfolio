Controller

The basic audio management in unity has a lot of good stuff in it. The mixer and the general augmentations that are available gives a lot of range. But when you start to implement it to your project there is a lot of code repetition. SoundKraft is created to help minimize the work of implementing sound. And furthermore it gives a more dynamic range of the sounds with a random range of the pitch and volume. By using a object pooled system we can keep the performance but have dynamic sound in game.

Pixelnary is a simple little game where you try to guess what the image that is slowly being de-pixelized is depicting.
